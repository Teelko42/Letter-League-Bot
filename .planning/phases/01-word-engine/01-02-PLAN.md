**src/engine/board.py** — Implement Board class:

```python
class Board:
    DEFAULT_ROWS = 19
    DEFAULT_COLS = 27

    def __init__(self, rows: int = DEFAULT_ROWS, cols: int = DEFAULT_COLS,
                 multiplier_layout: dict[tuple[int, int], MultiplierType] | None = None):
        self.rows = rows
        self.cols = cols
        self._multiplier_layout = multiplier_layout or {}
        self.grid: list[list[Cell]] = [
            [Cell(r, c, square_multiplier=self._multiplier_layout.get((r, c), MultiplierType.NONE))
             for c in range(cols)]
            for r in range(rows)
        ]

    def get_cell(self, row: int, col: int) -> Cell:
        if not (0 <= row < self.rows and 0 <= col < self.cols):
            raise IndexError(f"Cell ({row}, {col}) out of bounds for {self.rows}x{self.cols} board")
        return self.grid[row][col]

    def place_tile(self, row: int, col: int, letter: str, is_blank: bool = False,
                   mode: str = 'classic') -> None:
        cell = self.get_cell(row, col)
        cell.letter = letter.upper()
        cell.is_blank = is_blank
        if mode == 'wild':
            cell.bonded_multiplier = cell.square_multiplier

    def is_empty(self) -> bool:
        return all(cell.letter is None for row in self.grid for cell in row)

    @property
    def center(self) -> tuple[int, int]:
        return (self.rows // 2, self.cols // 2)
```

```python
def find_anchors(self, direction: str) -> list[tuple[int, int]]:
    """
    Find anchor squares for the given direction ('H' or 'V').
    An anchor is an empty cell adjacent to at least one occupied cell.
    On an empty board, return [self.center] as the only anchor.
    For horizontal: check left/right neighbors (and also above/below for any adjacency).
    For vertical: check above/below neighbors (and also left/right for any adjacency).
    An anchor is any empty cell that has at least one occupied orthogonal neighbor.
    """

def compute_cross_checks(self, gaddag: 'GADDAG', direction: str) -> dict[tuple[int, int], set[str]]:
    """
    For each empty cell, compute which letters can legally be placed there
    without creating an invalid perpendicular word.

    direction='H' means we're generating horizontal moves, so cross-checks
    validate VERTICAL perpendicular words (gather tiles above and below).

    direction='V' means we're generating vertical moves, so cross-checks
    validate HORIZONTAL perpendicular words (gather tiles left and right).

    If no perpendicular tiles exist for a cell, cross-check = all 26 letters.
    If perpendicular tiles exist, test each letter A-Z: form the perpendicular
    word and check gaddag.is_valid_word(). Only letters forming valid words
    are in the set.

    Returns: {(row, col): set[str]} for all empty cells.
    """

def _gather_perpendicular(self, row: int, col: int, direction: str) -> tuple[str, str] | None:
    """
    Gather the contiguous letters above/below (if direction='H') or
    left/right (if direction='V') of the given cell.
    Returns (prefix, suffix) strings, or None if no perpendicular tiles exist.
    """

def compute_left_limit(self, anchor_row: int, anchor_col: int, direction: str, rack_size: int) -> int:
    """
    How many cells to the left (H) or above (V) of the anchor can the
    LeftPart algorithm extend? Stops at: board edge, another anchor, or
    an occupied cell. Capped at rack_size - 1.
    """
```

Key implementation notes:
- Anchors are direction-independent in most implementations: any empty cell with at least one occupied orthogonal neighbor. Both H and V anchors use the same logic. The direction parameter exists for consistency but the anchor set is typically the same for both directions.
- Cross-checks ARE direction-dependent: H cross-checks validate vertical words, V cross-checks validate horizontal words. Two separate computations.
- `_gather_perpendicular` for direction='H': scan upward from (row-1, col) and downward from (row+1, col), collecting contiguous letters. Return (letters_above_reversed, letters_below). For direction='V': scan left and right.
- For cross-check word formation: the perpendicular word is prefix + candidate_letter + suffix.
- `compute_left_limit`: scan left (for H) or up (for V) from the cell before the anchor. Count empty cells until hitting an occupied cell, another anchor, or the board edge. Cap at rack_size - 1.
