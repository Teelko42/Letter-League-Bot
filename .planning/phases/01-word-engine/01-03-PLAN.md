```python
def score_word(cells_in_word: list[Cell], newly_placed_positions: set[tuple[int, int]],
               mode: str) -> tuple[int, int]:
    """
    Score a single word (main or perpendicular).

    cells_in_word: ordered list of Cell objects forming the word
    newly_placed_positions: set of (row, col) for tiles placed THIS turn
    mode: 'classic' or 'wild'

    Returns: (letter_sum_after_letter_multipliers, word_multiplier)

    Classic mode:
    - For each cell in the word:
      - Get tile_value = TILE_VALUES[cell.letter] if not cell.is_blank else 0
      - If (cell.row, cell.col) in newly_placed_positions:
        - Apply letter multiplier (DL: x2, TL: x3) to tile_value
        - Accumulate word multiplier (DW: x2, TW: x3, multiplicative)
      - Add tile_value to letter_sum
    - Return (letter_sum, accumulated_word_multiplier)

    Wild mode:
    - For each cell in the word:
      - Get tile_value = TILE_VALUES[cell.letter] if not cell.is_blank else 0
      - Apply cell.bonded_multiplier (DL: x2, TL: x3) to tile_value
      - Accumulate word multiplier from cell.bonded_multiplier (DW: x2, TW: x3)
      - ALSO: if (cell.row, cell.col) in newly_placed_positions AND cell has a
        square_multiplier that hasn't been bonded yet, apply it too
        (this handles the newly placed tile on this turn)
      - Add tile_value to letter_sum
    - Return (letter_sum, accumulated_word_multiplier)
    """

def score_move(board: 'Board', move_cells: list[Cell], newly_placed_positions: set[tuple[int, int]],
               perpendicular_words: list[list[Cell]], tiles_from_rack: int,
               rack_size: int, mode: str) -> ScoreBreakdown:
    """
    Score a complete move including main word, perpendicular words, and bingo bonus.

    1. Score main word: score_word(move_cells, newly_placed_positions, mode)
    2. Score each perpendicular word: score_word(perp, newly_placed_positions, mode)
    3. main_total = letter_sum * word_multiplier
    4. Apply bingo: if tiles_from_rack == rack_size, main_total *= 2
    5. total = main_total + sum(perpendicular scores)
    6. Return ScoreBreakdown with all components
    """
```

Key implementation rules (from CONTEXT.md and RESEARCH.md):
- Classic: square_multiplier applies ONLY on placement turn, ONLY to newly placed tiles
- Wild: bonded_multiplier applies EVERY turn to ALL tiles in the word
- Wild stacking: multiplicative (DW + DW = x4, TW + DW = x6). This is the research recommendation per open question #1.
- Blank tiles: ALWAYS 0 points, even on multiplier squares (DL of 0 = 0)
- Bingo: doubles main word ONLY (not perpendicular words). Letter League rule, NOT Scrabble +50.
- Bingo fires when ALL current rack tiles are consumed, regardless of rack count
- Perpendicular words: scored independently, each with their own multiplier evaluation

Do NOT import Board in scoring.py -- accept cells and positions as parameters to keep scoring pure and testable without board dependency.
