Create `src/vision/preprocessor.py` with a single public function:

```python
def preprocess_screenshot(img_bytes: bytes) -> bytes:
```

Implementation (follows 03-RESEARCH.md Pattern 2 exactly):

1. **Decode:** `np.frombuffer(img_bytes, np.uint8)` → `cv2.imdecode(nparr, cv2.IMREAD_COLOR)` → BGR image
2. **HSV conversion:** `cv2.cvtColor(bgr, cv2.COLOR_BGR2HSV)`
3. **Color mask:** `cv2.inRange(hsv, lower_bound, upper_bound)` — Use initial HSV bounds for the Letter League board background color. Start with a generous range and add a comment noting these need empirical calibration:
   ```python
   # HSV range for Letter League board background — needs empirical calibration
   # against real screenshots. These are initial estimates.
   BOARD_HSV_LOWER = np.array([15, 20, 160])
   BOARD_HSV_UPPER = np.array([35, 80, 240])
   ```
4. **Contour detection:** `cv2.findContours(mask, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)` → select largest by `cv2.contourArea`
5. **Area threshold:** If no contours found OR largest contour area < `MIN_BOARD_AREA` (10000 pixels), raise `VisNError(INVALID_SCREENSHOT, "Board region not detected — ...")`.
6. **Bounding rect:** `cv2.boundingRect(largest)` → `(x, y, w, h)`
7. **Pillow crop:** `Image.open(BytesIO(img_bytes)).crop((x, y, x+w, y+h))`
8. **2x upscale:** `cropped.resize((w*2, h*2), Image.Resampling.LANCZOS)`
9. **1568px clamp:** If `max(new_w, new_h) > 1568`, scale down proportionally to fit within 1568px on the long edge (per Claude Vision docs). Use `Image.Resampling.LANCZOS` for the downscale too.
10. **Output:** Save to `BytesIO` as PNG, return `.getvalue()`

Imports: `cv2`, `numpy as np`, `PIL.Image`, `io.BytesIO`, and `from src.vision.errors import VisNError, INVALID_SCREENSHOT`.

Add module-level constants: `MIN_BOARD_AREA = 10000`, `MAX_VISION_EDGE = 1568`, `UPSCALE_FACTOR = 2`.

Use `from __future__ import annotations`.

Do NOT apply any color or contrast adjustments (per user decision). Do NOT do any morphological operations or edge detection beyond the HSV color mask.
