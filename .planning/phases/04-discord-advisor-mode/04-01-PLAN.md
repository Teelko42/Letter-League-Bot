**`render_text_board(board: Board, move: Move, padding: int = 2) -> str`:**
Renders a windowed text-art view of the board around the placed word. Per user decision: "Top-1 move displayed as a text-art mock board showing letter placement on the grid."

Implementation:
1. Extract placed tiles (from_rack=True) and existing tiles from `move.tiles_used`.
2. Also check `board.grid[r][c].letter` for other existing tiles not in the move.
3. Compute bounding box of the move's tiles +/- `padding` cells, clamped to board edges.
4. Character set (Claude's discretion per CONTEXT.md):
   - Newly placed tile: `[X]` (brackets highlight the new placement)
   - Existing board tile: ` X ` (space-padded)
   - Empty cell: ` . ` (dot for empty)
5. Wrap output in monospace code block format (the caller wraps in triple backticks).
6. Add column/row indicators at edges for orientation.
7. Cap window to ~15 columns max to avoid Discord line-wrap.

**`build_success_embed(moves: list[Move], board: Board) -> discord.Embed`:**

**LetterLeagueBot class (subclasses commands.Bot):**
```python
class LetterLeagueBot(commands.Bot):
    def __init__(self, wordlist_path: Path, cache_path: Path | None = None) -> None:
        intents = discord.Intents.default()  # No privileged intents needed for slash-command-only bot
        super().__init__(command_prefix="!", intents=intents)
        self._wordlist_path = wordlist_path
        self._cache_path = cache_path
        self.gaddag: GADDAG | None = None
        self.difficulty_engine: DifficultyEngine | None = None
        self.channel_store = ChannelStore()

    async def setup_hook(self) -> None:
        """Load heavy resources and register commands before READY."""
        # GADDAG is CPU-bound — run in thread to avoid blocking event loop
        self.gaddag = await asyncio.to_thread(
            GADDAG.from_wordlist, self._wordlist_path, self._cache_path
        )
        self.difficulty_engine = DifficultyEngine()
        # Cog will be added by Plan 04-02
        # await self.add_cog(AdvisorCog(self))
        await self.tree.sync()
        logger.info("Bot setup complete — GADDAG loaded, commands synced")
```

Per research: Fail loudly if GADDAG loading fails. Don't catch the exception in setup_hook -- let it propagate and prevent the bot from starting.

**`main()` async function:**
```python
async def main() -> None:
    load_dotenv()
    token = os.environ.get("DISCORD_TOKEN")
    if not token:
        logger.error("DISCORD_TOKEN not set in environment. Add it to .env")

    # Wordlist and cache paths — configure via env or defaults

    bot = LetterLeagueBot(wordlist_path, cache_path)
    async with bot:
        await bot.start(token)
```
