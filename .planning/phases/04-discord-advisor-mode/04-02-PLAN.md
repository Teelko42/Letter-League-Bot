**src/bot/cog.py — AdvisorCog class:**

Create a `commands.Cog` subclass with three slash commands. Import everything needed from the plan 04-01 artifacts and existing engine/vision modules.

**Attachment validation constants:**
```python
ALLOWED_CONTENT_TYPES = {"image/png", "image/jpeg", "image/webp"}
MAX_ATTACHMENT_BYTES = 10_000_000  # 10 MB
```

**`/analyze` command handler:**
```python
@app_commands.command(name="analyze", description="Analyze a Letter League screenshot")
@app_commands.describe(screenshot="A screenshot of your Letter League game board")
async def analyze(self, interaction: discord.Interaction, screenshot: discord.Attachment) -> None:
```

Implementation flow — follow this EXACT order (per research pitfalls):

1. **Defer FIRST** (must be the absolute first line, per Pitfall 1):
   `await interaction.response.defer(ephemeral=True)`

2. **Get channel state:**
   `state = self.bot.channel_store.get(interaction.channel_id)`

3. **Validate attachment** (per Pitfall 4):
   - Check `screenshot.content_type` against ALLOWED_CONTENT_TYPES (use startswith since Discord may append charset)
   - Check `screenshot.size` against MAX_ATTACHMENT_BYTES
   - On failure: `await interaction.followup.send(embed=build_error_embed_generic("Please attach a PNG, JPEG, or WebP screenshot (max 10 MB)."))`
   - Return early on failure.

4. **Read attachment bytes:**
   `img_bytes = await screenshot.read()`

5. **Run vision pipeline** (already async, no thread needed):
   `board, rack = await extract_board_state(img_bytes, mode=state.mode)`

6. **Run engine** (CPU-bound sync, MUST use to_thread per Pitfall 2):
   `moves = await asyncio.to_thread(find_all_moves, board, rack, self.bot.gaddag, state.mode)`

7. **Handle no-moves case:**
   If `moves` is empty, send `build_no_moves_embed(board, rack)` and return.

8. **Apply difficulty selection** (CPU-bound sync, use to_thread):
   Only apply if difficulty < 100 (optimization: at 100%, the top move is always moves[0]).
   ```python
   if state.difficulty < 100:
       selected = await asyncio.to_thread(self.bot.difficulty_engine.select_move, moves, state.difficulty)
       # Reorder: put selected move first, then next 2 highest scoring
       top_moves = [selected] + [m for m in moves if m is not selected][:2]
   else:
       top_moves = moves[:3]
   ```

9. **Build and send success embed:**
   `embed = build_success_embed(top_moves, board)`
   `await interaction.followup.send(embed=embed)`

10. **Error handling (try/except around steps 4-9):**
    - `except VisNError as e:` -> `build_error_embed(e)`
    - `except Exception:` -> log with `logger.exception(...)`, send `build_error_embed_generic()`
    - Always send via `interaction.followup.send(embed=..., ephemeral=True)` (NOT `interaction.response.send_message` — per Pitfall: InteractionAlreadyResponded after defer)
