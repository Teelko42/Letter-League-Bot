**src/browser/session.py:**

Create a `BrowserSession` class with these methods:

1. `__init__(self, profile_dir: str = "./browser_data")` — store profile path, initialize pw/context/page as None.

2. `async def start(self) -> Page` — Main entry point:
   - Detect first-run vs returning using `_profile_exists()` (check for `{profile_dir}/Default/Cookies` file).
   - Call `_launch(headless=not is_first_run)` to get the patchright persistent context.
   - Get or create a page from the context (`context.pages[0]` if exists, else `context.new_page()`).
   - If first-run: call `_first_run_login(page)`.
   - If returning: call `_validate_session(page)`. If invalid, log error with `logger.error()` explaining the session has expired and the operator should delete `./browser_data/` and re-run, then call `await self.close()` and `sys.exit(1)`.
   - Return the page.

3. `async def _launch(self, headless: bool) -> None` — Launch patchright:
   ```python
   from patchright.async_api import async_playwright
   self._pw = await async_playwright().start()
   self._context = await self._pw.chromium.launch_persistent_context(
       user_data_dir=self._profile_dir,
       headless=headless,
       viewport={"width": 1280, "height": 800},
   )
   ```
   Use exactly these viewport dimensions (1280x800) per research recommendation.
   Do NOT pass custom user_agent or extra chromium args (patchright handles stealth).

4. `def _profile_exists(self) -> bool` — Check `Path(self._profile_dir) / "Default" / "Cookies"` exists.

5. `async def _first_run_login(self, page) -> None`:
   - Navigate to `https://discord.com/login`.
   - Log: "First run detected — please log into Discord in the browser window."
   - Wait for login completion selector `[data-list-id="guildsnav"]` with 5-minute timeout (300_000 ms).
   - On success, log: "Discord login detected — session saved."
   - On timeout, log error: "Login timed out after 5 minutes." then close and exit.

6. `async def _validate_session(self, page) -> bool`:
   - Navigate to `https://discord.com/channels/@me` with `wait_until="domcontentloaded"`.
   - Return `"login" not in page.url`.

7. `async def close(self) -> None` — Close context and stop playwright (with None checks).

**Key patterns to follow:**
- `from __future__ import annotations` at top (project convention).
- Use `from loguru import logger` for all logging (no print statements).
- Use `from pathlib import Path` for file path checks.
- Import `sys` for `sys.exit(1)` on fatal errors.
- Type hints on all method signatures.
- No `try/except` around `launch_persistent_context` — let patchright's own error messages propagate (they are descriptive, e.g., locked profile directory).
