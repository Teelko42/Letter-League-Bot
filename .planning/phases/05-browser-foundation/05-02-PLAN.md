**src/browser/capture.py:**

Create two public functions:

1. **`def is_non_blank(img_bytes: bytes, threshold: float = 5.0) -> bool`**
   - Decode PNG bytes to numpy array via `cv2.imdecode`.
   - Return `True` if `np.std(img) > threshold`.
   - Return `False` if `img is None` (decode failed) or variance below threshold.
   - This is a quick pre-check before the expensive vision pipeline call.

2. **`async def capture_canvas(page, frame, max_retries: int = 3) -> bytes`**
   - **Wait for render**: `await page.wait_for_load_state("networkidle")` then `await asyncio.sleep(1.0)` for canvas render buffer.
   - **Primary capture method**: Use `page.frame_locator('iframe[src*="discordsays.com"]')` to get a FrameLocator, then `.locator("canvas").first` to get the canvas. Call `await canvas.screenshot()` to get bytes.
   - **Fallback capture method**: If primary raises an exception, fall back to `await frame.evaluate("document.querySelector('canvas').toDataURL('image/png')")`, extract the base64 portion after the comma, and decode with `base64.b64decode()`. Log a warning when falling back.
   - **Non-blank check**: Call `is_non_blank(screenshot_bytes)`. If blank, retry up to `max_retries` times with a 2-second wait between attempts. Log each retry: "Screenshot blank (attempt {n}/{max}), retrying...". If all attempts return blank, raise `RuntimeError("Canvas screenshot is blank after {max_retries} attempts")`.
   - **Return** the non-blank screenshot bytes.

**Do NOT call `extract_board_state()` inside `capture_canvas()`** — keep capture and vision separate. The entry-point script below calls both in sequence. This keeps capture.py independent of the vision pipeline for reuse in later phases.
