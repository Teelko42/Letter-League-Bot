Create `src/browser/turn_detector.py` with the following structure. Use `from __future__ import annotations` (project convention).

**TurnState type:**
```python
from typing import Literal
TurnState = Literal["my_turn", "not_my_turn", "game_over"]
```

**Constants (all marked with `# PLACEHOLDER — calibrate from live screenshots`):**
- `BANNER_HSV_LOWER = np.array([5, 120, 150])` — orange banner lower bound
- `BANNER_HSV_UPPER = np.array([20, 255, 255])` — orange banner upper bound
- `BANNER_ROI_FRAC = (0.0, 0.15)` — top 15% of canvas height (fractional, not absolute px)
- `BANNER_CONFIDENCE = 0.10` — 10% of ROI pixels must be orange
- `GAME_OVER_BOARD_THRESHOLD = 0.02` — threshold for game-over detection (reduced board visibility)
- `POLL_FAST_S = 1.5` — fast polling interval (1-2s range per CONTEXT.md)
- `POLL_SLOW_S = 5.0` — slow polling interval after idle
- `IDLE_THRESHOLD_S = 30.0` — seconds before switching to slow polling
- `_DEBUG_DIR = Path(__file__).parent.parent.parent / "debug" / "turn_detection"` — relative to project root

**`_save_debug_screenshot(img_bytes: bytes, label: str = "") -> Path`:**
- Create `_DEBUG_DIR` with `mkdir(parents=True, exist_ok=True)`
- Write bytes to `frame_{timestamp}_{label}.png` using `datetime.now().strftime("%Y%m%d_%H%M%S_%f")`
- Return the Path

**`_is_my_turn(img_bytes: bytes) -> bool`:**
- Decode bytes to BGR via `cv2.imdecode(np.frombuffer(img_bytes, dtype=np.uint8), cv2.IMREAD_COLOR)`
- Return False if decode fails
- Crop ROI: top `BANNER_ROI_FRAC[0]` to `BANNER_ROI_FRAC[1]` of image height, full width
- Convert ROI to HSV via `cv2.cvtColor(roi, cv2.COLOR_BGR2HSV)`
- Create mask via `cv2.inRange(hsv, BANNER_HSV_LOWER, BANNER_HSV_UPPER)`
- Compute `orange_ratio = np.count_nonzero(mask) / mask.size`
- Return `orange_ratio >= BANNER_CONFIDENCE`

**`_is_game_over(img_bytes: bytes) -> bool`:**
- Decode bytes to BGR (same pattern)
- Return False if decode fails
- Use the board peach color constants from `src.vision.preprocessor`: `BOARD_HSV_LOWER`, `BOARD_HSV_UPPER`
- Check the center region of the frame (middle 50% height, middle 50% width) for board peach color
- If the peach ratio in the center is very low (below `GAME_OVER_BOARD_THRESHOLD`) while the overall image is non-blank, the leaderboard overlay is likely occluding the board center — return True
- Also check: if the top banner ROI has NO orange (not my turn) AND the center has minimal peach, it's game over
- Return False otherwise (normal gameplay state)
- NOTE: This is a heuristic; exact thresholds will be calibrated in Plan 02

**`classify_frame(img_bytes: bytes) -> TurnState`:**
- Check game_over FIRST (per research — prevents infinite polling after game ends)
- Then check my_turn
- Default to not_my_turn

**`async def preflight_check(page: Any) -> None`:**
- Import `capture_canvas` from `src.browser.capture`
- Call `capture_canvas(page)` to get one frame
- Call `classify_frame(img_bytes)` to verify no crash
- Save screenshot via `_save_debug_screenshot(img_bytes, label="preflight")`
- Log: `"Pre-flight turn detection check passed — initial state: {state}"`
- If capture_canvas raises, re-raise as RuntimeError with clear message

**`async def poll_turn(page: Any) -> TurnState`:**
- Adaptive polling loop per CONTEXT.md decisions:
  - Track `last_state: TurnState | None` and `idle_duration: float`
  - Capture screenshot via `capture_canvas(page)` with try/except — on failure, log warning and retry with exponential backoff (base 1.0s, cap 30.0s), then `continue`
  - Reset capture backoff on successful capture
  - Classify frame via `classify_frame(img_bytes)`
  - Log ONLY on state change: `"Turn state changed: {last_state} -> {state}"` (quiet logging per CONTEXT.md)
  - Reset `idle_duration = 0.0` on state change
  - Return immediately if state is `"my_turn"` or `"game_over"`
  - Adaptive backoff: track `idle_duration`; if `>= IDLE_THRESHOLD_S`, use `POLL_SLOW_S`; else use `POLL_FAST_S`
  - Increment idle_duration by the interval used
  - `await asyncio.sleep(interval)`

**Update `src/browser/__init__.py`:**
- Add import: `from src.browser.turn_detector import TurnState, classify_frame, poll_turn, preflight_check`
- Add to `__all__`: `"TurnState"`, `"classify_frame"`, `"poll_turn"`, `"preflight_check"`

Create `scripts/calibrate_turn.py` — an interactive calibration script the operator runs during a live Letter League game to collect screenshots and discover HSV thresholds for turn detection.

Use `from __future__ import annotations` (project convention). Use `loguru` logger.

**Structure:**

```python
"""Calibration script for turn detection HSV thresholds.

Run during a live Letter League game to capture screenshots and print
HSV statistics for the banner region. Saves frames to debug/turn_detection/
for visual inspection and threshold tuning.

Usage:
    python scripts/calibrate_turn.py

Requires DISCORD_CHANNEL_URL in .env file.
Press Ctrl+C to stop capturing.
"""
```

**Imports:** asyncio, os, sys, datetime, pathlib.Path, cv2, numpy, loguru.logger, dotenv.load_dotenv, BrowserSession, navigate_to_activity, capture_canvas from src.browser

**Constants:**
- `DEBUG_DIR = Path("debug/turn_detection")` — relative to CWD (project root)
- `BANNER_ROI_FRAC = (0.0, 0.15)` — same as turn_detector.py
- `CAPTURE_INTERVAL = 2.0` — seconds between captures

**`async def main() -> None:`**

1. `load_dotenv()`
2. Read `DISCORD_CHANNEL_URL` from env; exit with error if missing
3. Create `DEBUG_DIR` with `mkdir(parents=True, exist_ok=True)`
4. Start browser session: `session = BrowserSession()`, `page = await session.start()`
5. Navigate to activity: `await navigate_to_activity(page, channel_url)`
6. Log: `"Starting calibration — capturing every {CAPTURE_INTERVAL}s. Press Ctrl+C to stop."`
7. Frame capture loop (wrapped in try/except KeyboardInterrupt):
   - `count = 0`
   - While True:
     a. `img_bytes = await capture_canvas(page)`
     b. Save to `debug/turn_detection/frame_{timestamp}.png`
     c. Decode to BGR, crop banner ROI (same fractional crop as turn_detector)
     d. Convert ROI to HSV
     e. Print per-frame stats:
        - Median H, S, V of entire ROI
        - Min H, S, V of entire ROI
        - Max H, S, V of entire ROI
        - Orange pixel ratio using the PLACEHOLDER thresholds from turn_detector (import them)
     f. Also analyze center region (middle 50%) for game-over calibration:
        - Peach pixel ratio using BOARD_HSV_LOWER/UPPER from preprocessor
     g. Log all stats in a single structured log line per frame
     h. Increment count
     i. `await asyncio.sleep(CAPTURE_INTERVAL)`
8. On KeyboardInterrupt: Log `"Calibration stopped. {count} frames saved to {DEBUG_DIR}"`
9. Finally: `await session.close()`

**Entry point:** `if __name__ == "__main__": asyncio.run(main())`

**Key details:**
- Import `BANNER_HSV_LOWER, BANNER_HSV_UPPER, BANNER_ROI_FRAC, BANNER_CONFIDENCE` from `src.browser.turn_detector` so the calibration script uses the same constants it's calibrating
- Import `BOARD_HSV_LOWER, BOARD_HSV_UPPER` from `src.vision.preprocessor` for game-over peach ratio
- Print a summary at the end with suggested threshold ranges based on observed min/max values across all captured frames
