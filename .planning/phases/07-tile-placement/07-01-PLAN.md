**Module-level fractional constants (PLACEHOLDER values — calibration script in Task 2 will provide real values):**
```python
# All fractions relative to canvas width or height.
# PLACEHOLDER — must be calibrated from live game screenshots.
GRID_X0_FRAC = 0.03      # left edge of board grid (fraction of canvas width)
GRID_Y0_FRAC = 0.02      # top edge of board grid (fraction of canvas height)
CELL_W_FRAC = 0.034       # one cell width (fraction of canvas width)
CELL_H_FRAC = 0.049       # one cell height (fraction of canvas height)
RACK_Y_FRAC = 0.92        # rack row vertical center (fraction of canvas height)
RACK_X0_FRAC = 0.15       # first rack tile center (fraction of canvas width)
RACK_TILE_STEP_FRAC = 0.035  # spacing between rack tile centers (fraction of width)
CONFIRM_X_FRAC = 0.87     # confirm button center X (fraction of canvas width)
CONFIRM_Y_FRAC = 0.95     # confirm button center Y (fraction of canvas height)
```

**Custom exception:**
```python
class PlacementError(Exception):
    """Raised when tile placement fails after retries."""
```

**CoordMapper class:**
- `__init__(self, bbox: dict)` — stores the bounding box dict (`{x, y, width, height}`). Raises `PlacementError` if bbox is None.
- `board_cell_px(self, row: int, col: int) -> tuple[float, float]` — returns viewport pixel coordinates for the center of board cell (row, col).
  - Formula: `x = bbox["x"] + (GRID_X0_FRAC + col * CELL_W_FRAC) * bbox["width"]`
  - Formula: `y = bbox["y"] + (GRID_Y0_FRAC + row * CELL_H_FRAC) * bbox["height"]`
- `rack_tile_px(self, slot_index: int) -> tuple[float, float]` — returns viewport pixel coordinates for rack tile at given slot index.
  - Formula: `x = bbox["x"] + (RACK_X0_FRAC + slot_index * RACK_TILE_STEP_FRAC) * bbox["width"]`
  - Formula: `y = bbox["y"] + RACK_Y_FRAC * bbox["height"]`
- `confirm_btn_px(self) -> tuple[float, float]` — returns viewport pixel coordinates for the confirm button.

**Helper functions:**
- `jitter(x: float, y: float, px: int = 3) -> tuple[float, float]` — add uniform random jitter within +/-px to both coordinates.
- `assign_rack_indices(rack: list[str], rack_tiles: list[TileUse]) -> list[int]` — map each TileUse to leftmost available rack slot index. When `tile.is_blank` is True, search for `'?'` in the rack (not `tile.letter`). Raise `ValueError` if a tile letter is not found in remaining rack slots.

**TilePlacer class:**
- `__init__(self, page: Any)` — stores the patchright Page object.
- `async _get_canvas_bbox(self) -> dict` — get canvas bounding box via `page.frame_locator('iframe[src*="discordsays.com"]').locator("canvas").first.bounding_box()`. Raise `PlacementError` if None.
- `async _drag_tile(self, from_x, from_y, to_x, to_y, steps: int = 10)` — execute mouse drag sequence: `page.mouse.move(from_x, from_y)` -> `page.mouse.down()` -> `asyncio.sleep(0.05)` -> `page.mouse.move(to_x, to_y, steps=steps)` -> `page.mouse.up()`.
- `async _verify_placement(self, before_bytes: bytes) -> bool` — capture a new screenshot via `capture_canvas(self.page)`, decode both images with OpenCV, compute mean absolute pixel difference, return True if diff > 1.0 (tile landed). Return False if either image fails to decode.
- `async place_tiles(self, move: Move, rack: list[str]) -> None` — orchestrate placement of all rack tiles for a move:
  1. Call `_get_canvas_bbox()` and create a `CoordMapper`.
  2. Get rack tiles via `move.rack_tiles_consumed()`.
  3. Map tiles to slot indices via `assign_rack_indices(rack, rack_tiles)`.
  4. Sort placements by word-spelling order: by `col` ascending if `move.direction == 'H'`, by `row` ascending if `move.direction == 'V'`.
  5. For each tile placement:
     a. Compute rack source coordinates with jitter: `jitter(*mapper.rack_tile_px(slot_idx))`.
     b. Compute board target coordinates with jitter: `jitter(*mapper.board_cell_px(tile.row, tile.col))`.
     c. Capture before-screenshot for verification.
     d. Execute `_drag_tile(rx, ry, bx, by)`.
     e. Call `_verify_placement(before_bytes)` — if False, retry the drag once. If second attempt also fails, raise `PlacementError`.
     f. `await asyncio.sleep(random.uniform(1.0, 3.0))` for human-like delay between placements.

Use `from __future__ import annotations` at the top. Import `from src.browser.capture import capture_canvas` for verification screenshots. Import `Move` and `TileUse` under `TYPE_CHECKING` guard. Use loguru for logging (log each drag attempt: tile letter, from/to coordinates, success/retry).
